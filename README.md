# Resume-Analyzer
Resume Analyzer is a Python-based web application that extracts and analyzes resumes and compares them with job descriptions to provide skill matching, improvement suggestions, feedback and scoring to help users improve their CV and match job requirements more effectively..

----

## Features
- Extract text from PDF and DOCX resumes
- Skill extraction and matching
- Job description comparison
- Resume scoring system
- Simple web interface (Flask)

----

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS

---

## ▶️ How to Run

```bash
git clone https://github.com/your-username/resume-analyzer.git
cd resume-analyzer
pip install -r requirements.txt
python app.py
```
## Then open:
   http://127.0.0.1:5000/
